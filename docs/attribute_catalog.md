# Project G — 속성 카탈로그

> 작성일: 2026-02-20
> 상태: 초안

게임에서 사용되는 모든 열거형 속성과 유효값 목록.
게임메이커 UI의 드롭다운/선택지 원본이자, ContentRegistry 구현 시 Rust enum의 근거 문서.

> 각 속성의 상세 사용 맥락은 `docs/entity_definition.md`와 `docs/database_design.md`의 JSON 예시 참조.

---

## 1. 게임 설정

### mode — 게임 모드

게임이 MUD(텍스트 기반)인지 2D(그리드 기반)인지 결정한다.

| 값 | 설명 |
|----|------|
| mud | 텍스트 MUD — 방(Room) 기반 이산 공간, Telnet/WebSocket 접속 |
| grid | 2D 그리드 — 타일 기반 좌표 공간, WebSocket 접속 |

### auth_mode — 인증 방식

계정과 캐릭터의 관계를 결정한다.

| 값 | 설명 |
|----|------|
| character | 캐릭터 = 계정 — 이름과 비밀번호로 바로 접속 (클래식 MUD) |
| account | 계정 + 캐릭터 분리 — 로그인 후 캐릭터 선택 (MMO 방식) |

## 2. 엔티티 분류

### entity_kind — 엔티티 종류

스폰 규칙, 순찰 경로 등에서 어떤 종류의 엔티티를 참조하는지 구분한다.

| 값 | 설명 |
|----|------|
| monster | 몬스터 — 적대적 전투 대상 |
| npc | NPC — 비전투 캐릭터 (상인, 교관, 퀘스트 발주자 등) |
| item | 아이템 — 바닥에 놓인 습득 가능 아이템 |
| object | 오브젝트 — 방/맵에 설치된 상호작용 대상 (문, 상자, 표지판 등) |
| pet | 펫 — 플레이어 소유 동반자 |

### object_kind — 오브젝트 세부 종류

오브젝트의 구체적인 타입을 구분한다. 타입에 따라 사용 가능한 속성이 달라진다.

| 값 | 설명 |
|----|------|
| door | 문 — 열림/닫힘/잠김 상태, 통과 조건 |
| chest | 상자 — 아이템 보관, 잠금 가능 |
| trap | 함정 — 숨겨진 피해/상태이상 발동 |
| sign | 표지판 — 읽기 텍스트 표시 |
| campfire | 모닥불 — 범위 내 HP 회복 효과 |
| crafting_station | 제작대 — 해당 제작 분류의 레시피 사용 가능 |
| bulletin_board | 게시판 — 플레이어가 글을 읽고 쓸 수 있음 |

### item_kind — 아이템 종류

아이템의 사용 방식과 적용 가능한 속성을 결정한다.

| 값 | 설명 |
|----|------|
| weapon | 무기 — 장비 슬롯(무기)에 착용, 공격력 보너스 |
| armor | 방어구 — 장비 슬롯(머리/몸통/다리/발/손)에 착용, 방어력 보너스 |
| accessory | 악세서리 — 장비 슬롯(악세서리1/2)에 착용, 특수 효과 |
| consumable | 소비품 — 사용 시 효과 발동 (회복, 버프 부여), 사용 후 소멸 |
| material | 재료 — 제작에 사용되는 소재 |
| quest | 퀘스트 아이템 — 특정 퀘스트 전용, 거래/드롭 불가 |
| key | 열쇠 — 잠긴 문/상자를 여는 데 사용 |
| container | 가방 — 인벤토리 확장, 내부에 아이템 보관 |
| currency | 화폐 — 골드, 특수 재화 등 중첩 가능한 교환 매체 |

## 3. 등급/희귀도

### grade — 몬스터 등급

몬스터의 강함과 보상 수준을 나타낸다. 등급이 높을수록 HP/공격력이 높고 드롭이 좋다.

| 값 | 설명 |
|----|------|
| normal | 일반 — 기본 등급, 솔로 처치 가능 |
| elite | 정예 — 강화된 개체, 더 좋은 보상 |
| boss | 보스 — 지역/던전 보스, 소규모 파티 권장 |
| raid | 레이드 — 대규모 파티 필요, 최고급 보상 |

### rarity — 아이템 희귀도

아이템의 품질 등급. 높을수록 성능이 좋고 획득이 어렵다.

| 값 | 설명 |
|----|------|
| common | 일반 — 흔한 아이템, 상점 판매 |
| uncommon | 고급 — 약간 희귀, 특정 몬스터 드롭 |
| rare | 희귀 — 낮은 드롭률, 눈에 띄는 성능 |
| epic | 영웅 — 매우 희귀, 보스/퀘스트 보상 |
| legendary | 전설 — 최고 등급, 서버 내 소수만 보유 |

## 4. 캐릭터/NPC

### role — NPC 역할

NPC의 주요 기능을 결정한다. 역할에 따라 사용 가능한 하위 데이터가 달라진다.

| 값 | 설명 |
|----|------|
| merchant | 상인 — 아이템 매매 (shop 참조) |
| trainer | 교관 — 스킬 습득 (trainable_skills 참조) |
| quest_giver | 퀘스트 발주자 — 퀘스트 수락/완료 (quest_offers 참조) |
| banker | 은행원 — 아이템 보관 서비스 |
| guard | 경비병 — 특정 구역 순찰/경비 |
| generic | 일반 — 특수 기능 없음, 대화만 가능 |

### direction — 바라보는 방향 (2D)

2D 모드에서 캐릭터/NPC의 스프라이트 방향.

| 값 | 설명 |
|----|------|
| up | 위 — 화면 상단을 향함 |
| down | 아래 — 화면 하단을 향함 (기본값) |
| left | 왼쪽 |
| right | 오른쪽 |

## 5. 장비

### slot — 장비 슬롯

방어구/악세서리가 장착되는 신체 부위.

| 값 | 대상 | 설명 |
|----|------|------|
| head | 방어구 | 머리 — 투구, 모자 |
| body | 방어구 | 몸통 — 갑옷, 로브 |
| legs | 방어구 | 다리 — 바지, 각반 |
| feet | 방어구 | 발 — 부츠, 신발 |
| hands | 방어구 | 손 — 장갑, 건틀릿 |
| acc1 | 악세서리 | 악세서리 슬롯 1 — 반지, 목걸이 등 |
| acc2 | 악세서리 | 악세서리 슬롯 2 |

### weapon_type — 무기 종류

무기의 형태. 클래스별 장비 가능 무기 제한에 사용된다.

| 값 | 설명 |
|----|------|
| sword | 검 — 균형 잡힌 근접 무기 |
| axe | 도끼 — 높은 공격력, 느린 속도 |
| mace | 둔기 — 방어 관통, 중간 속도 |
| dagger | 단검 — 낮은 공격력, 빠른 속도, 높은 크리티컬 |
| bow | 활 — 원거리 물리 공격 |
| staff | 지팡이 — 원거리 마법 공격, 높은 마법 공격력 |
| wand | 완드 — 한손 마법 무기, 방패 병용 가능 |

### equip_armor_weight — 장비 가능 방어구 무게

클래스가 착용할 수 있는 방어구의 최대 무게 등급. 해당 등급 이하의 방어구를 모두 장비 가능.

| 값 | 설명 |
|----|------|
| cloth | 천 — 마법사 계열 (천 방어구만 착용) |
| leather | 가죽 — 도적/궁수 계열 (천 + 가죽) |
| mail | 사슬 — 성직자/레인저 계열 (천 + 가죽 + 사슬) |
| plate | 판금 — 전사 계열 (모든 방어구 착용) |

## 6. 속성/원소

### element — 원소 속성

무기, 스킬, 몬스터 등에 부여되는 원소. 속성 상성(유리/불리)에 영향.

| 값 | 설명 |
|----|------|
| none | 무속성 — 상성 없음 (기본값) |
| fire | 화염 — 지속 데미지(화상)와 시너지 |
| ice | 냉기 — 이동 속도 감소와 시너지 |
| lightning | 번개 — 스턴 효과와 시너지 |
| poison | 독 — 지속 데미지(중독)와 시너지 |
| holy | 신성 — 언데드에 추가 데미지, 힐 강화 |
| dark | 암흑 — 생명력 흡수와 시너지 |

## 7. 전투/스킬

### skill type — 스킬 타입

스킬의 사용 방식을 구분한다.

| 값 | 설명 |
|----|------|
| active | 액티브 — 직접 사용, MP 소모, 쿨다운 있음 |
| passive | 패시브 — 습득 시 상시 적용, 사용 불필요 |

### target_type — 스킬 대상 타입

스킬이 어떤 대상에게 적용되는지 결정한다.

| 값 | 설명 |
|----|------|
| self | 자기 자신 — 자가 버프/힐 |
| single_enemy | 적 단일 — 단일 대상 공격/디버프 |
| single_ally | 아군 단일 — 단일 대상 힐/버프 |
| aoe_enemy | 적 범위 — 반경 내 모든 적 |
| aoe_ally | 아군 범위 — 반경 내 모든 아군 |
| aoe_all | 전체 범위 — 반경 내 적/아군 모두 |

### effect_type — 스킬 효과 타입

스킬이 대상에게 주는 효과의 종류.

| 값 | 설명 |
|----|------|
| damage | 데미지 — 즉시 HP 감소 |
| heal | 힐 — 즉시 HP 회복 |
| buff | 버프 부여 — 대상에게 긍정적 상태 효과 |
| debuff | 디버프 부여 — 대상에게 부정적 상태 효과 |
| dot | 지속 데미지 — 일정 시간 동안 주기적 HP 감소 |
| hot | 지속 회복 — 일정 시간 동안 주기적 HP 회복 |
| dispel | 해제 — 대상의 버프/디버프 제거 |
| teleport | 텔레포트 — 대상을 특정 위치로 이동 |
| summon | 소환 — 동반자/오브젝트 생성 |

### scaling_stat — 스케일링 스탯

스킬 효과의 수치가 어떤 스탯에 비례해서 증가하는지 결정한다.

| 값 | 설명 |
|----|------|
| none | 스케일링 없음 — 고정 수치만 적용 |
| attack | 물리 공격력 비례 — 전사/궁수 스킬 |
| magic_attack | 마법 공격력 비례 — 마법사 스킬 |

## 8. 버프/상태 효과

### buff category — 버프 분류

상태 효과가 긍정적인지 부정적인지 구분한다.

| 값 | 설명 |
|----|------|
| buff | 버프 — 긍정적 효과 (아군이 부여) |
| debuff | 디버프 — 부정적 효과 (적이 부여, 해제 대상) |

### buff effect_type — 버프 효과 타입

버프/디버프가 대상에게 주는 구체적인 효과.

| 값 | 설명 |
|----|------|
| stat_modify | 스탯 변경 — 공격력/방어력 등 수치 증감 |
| dot | 지속 데미지 — 주기적 HP 감소 (독, 화상 등) |
| hot | 지속 회복 — 주기적 HP 회복 (재생 등) |
| immunity | 면역 — 특정 상태이상에 걸리지 않음 |
| speed_modify | 속도 변경 — 이동/공격 속도 증감 |
| stun | 기절 — 행동 불가 |
| silence | 침묵 — 스킬 사용 불가 |
| root | 속박 — 이동 불가 (공격/스킬은 가능) |

## 9. AI

### ai_type — AI 행동 타입

몬스터/NPC의 기본 행동 패턴을 결정한다.

| 값 | 설명 |
|----|------|
| passive | 비선공 — 공격받기 전까지 행동하지 않음 |
| aggressive | 선공 — 감지 범위 내 플레이어를 자동 공격 |
| patrol | 순찰 — 지정된 경유지를 따라 이동, 감지 시 전투 |
| stationary | 고정 — 위치에서 움직이지 않음, 범위 내 적만 공격 |
| scripted | 스크립트 — Lua 스크립트로 커스텀 AI 제어 |

### attack_style — 공격 방식

몬스터의 기본 공격 형태.

| 값 | 설명 |
|----|------|
| melee | 근접 — 인접 거리에서 물리 공격 |
| ranged | 원거리 — 투사체 기반 물리 공격 |
| magic | 마법 — 마법 기반 원거리 공격 |
| hybrid | 복합 — 거리에 따라 근접/원거리 전환 |

### target_priority — 타겟 우선순위

여러 대상 중 어떤 플레이어를 우선 공격할지 결정한다.

| 값 | 설명 |
|----|------|
| nearest | 가장 가까운 대상 (기본값) |
| lowest_hp | 현재 HP가 가장 낮은 대상 |
| highest_damage | 자신에게 가장 많은 피해를 준 대상 |
| random | 무작위 선택 |

## 10. 공간/이동

### exit direction — 출구 방향 (MUD)

MUD 모드에서 방과 방을 연결하는 이동 방향.

| 값 | 설명 |
|----|------|
| north | 북쪽 |
| south | 남쪽 |
| east | 동쪽 |
| west | 서쪽 |
| up | 위 — 계단, 사다리 등 |
| down | 아래 — 지하, 절벽 등 |

### door initial_state — 문 초기 상태

문 오브젝트의 게임 시작 시 상태.

| 값 | 설명 |
|----|------|
| open | 열림 — 자유롭게 통과 가능 |
| closed | 닫힘 — 열기 동작 필요 |
| locked | 잠김 — 열쇠 필요 |

### trigger_type — 함정 발동 방식

함정 오브젝트가 어떤 조건에서 발동되는지 결정한다.

| 값 | 설명 |
|----|------|
| step | 밟기 — 타일 위를 지나가면 발동 |
| proximity | 근접 — 일정 거리 내 접근 시 발동 |
| timed | 시간 — 주기적으로 자동 발동 |

## 11. 퀘스트

### objective_type — 퀘스트 목표 타입

퀘스트 단계의 달성 조건 종류.

| 값 | 설명 |
|----|------|
| kill | 처치 — 특정 몬스터 N마리 처치 |
| collect | 수집 — 특정 아이템 N개 획득 |
| visit | 방문 — 특정 방/좌표에 도달 |
| talk | 대화 — 특정 NPC와 대화 |
| use | 사용 — 특정 아이템/오브젝트 사용 |
| escort | 호위 — NPC를 목적지까지 호위 |
| craft | 제작 — 특정 아이템 N개 제작 |

### quest status — 퀘스트 진행 상태 (런타임)

플레이어의 퀘스트 진행 상태. save_data에 저장된다.

| 값 | 설명 |
|----|------|
| active | 진행 중 — 현재 수행 중인 퀘스트 |
| completed | 완료 — 모든 단계를 마치고 보상을 받음 |
| failed | 실패 — 시간 초과 등으로 실패 |

## 12. 제작

### craft_type — 제작 분류

레시피와 제작대의 종류를 구분한다. 레시피의 craft_type과 제작대의 craft_type이 일치해야 제작 가능.

| 값 | 설명 |
|----|------|
| forge | 대장간 — 무기, 방어구, 금속 도구 |
| alchemy | 연금술 — 물약, 독약, 마법 재료 |
| cooking | 요리 — 음식, 음료 (소비품) |

---

## 13. 업적

### condition_type — 업적 달성 조건 타입

업적의 달성 조건 종류.

| 값 | 설명 |
|----|------|
| kill_count | 처치 수 — 특정 몬스터(또는 전체) N마리 처치 |
| collect_count | 수집 수 — 특정 아이템 N개 보유/획득 |
| quest_clear | 퀘스트 완료 — 특정 퀘스트 클리어 |
| level_reach | 레벨 도달 — 캐릭터 레벨 N 달성 |
| visit | 방문 — 특정 장소 방문 |
| craft_count | 제작 수 — 아이템 N개 제작 |
| play_time | 플레이 시간 — 누적 접속 시간 N초 이상 |

## 14. 맵/환경 (2D)

### event_type — 맵 트리거 이벤트 타입

맵 트리거 영역에 진입했을 때 발동되는 이벤트의 종류.

| 값 | 설명 |
|----|------|
| script | 스크립트 — Lua 스크립트 실행 |
| cutscene | 컷신 — 연출 재생 |
| dialogue | 대화 — 대화 트리 시작 |
| combat | 전투 — 강제 전투 진입 |

### environment type — 환경 오브젝트 타입

맵에 배치되는 환경 효과의 종류.

| 값 | 설명 |
|----|------|
| light | 광원 — 빛을 발산하는 오브젝트 (횃불, 램프 등) |
| weather | 날씨 — 비, 눈, 안개 등 영역 효과 |
| particle | 파티클 — 시각 효과 (불꽃, 먼지, 반딧불 등) |
| sound | 사운드 — 환경 음향 (강물 소리, 바람 등) |

### weather_type — 날씨 종류

날씨 환경 오브젝트의 구체적 타입.

| 값 | 설명 |
|----|------|
| rain | 비 — 시야 감소, 화염 약화 |
| snow | 눈 — 이동 속도 감소 |
| fog | 안개 — 시야 대폭 감소 |
| storm | 폭풍 — 비 + 번개, 주기적 데미지 가능 |
| sandstorm | 모래폭풍 — 시야/속도 감소, 지속 데미지 |

## 16. 공통 속성 설명

JSON 파일 전반에서 반복 사용되는 범용 속성.

| 속성 | 타입 | 설명 |
|------|------|------|

| name | 문자열 | 게임 내 표시 이름 |
| description | 문자열 | 상세 설명 텍스트 |

### 전투 스탯

| 속성 | 타입 | 설명 |
|------|------|------|
| hp_max | 정수 | 최대 체력 |
| mp_max | 정수 | 최대 마나 |
| attack | 정수 | 물리 공격력 |
| defense | 정수 | 물리 방어력 |
| magic_attack | 정수 | 마법 공격력 |
| magic_defense | 정수 | 마법 방어력 |
| speed | 정수 | 행동/이동 속도 |
| critical_rate | 실수 | 크리티컬 확률 (0.0~1.0) |
| critical_damage | 실수 | 크리티컬 데미지 배율 (기본 1.5) |

### 아이템 공통

| 속성 | 타입 | 설명 |
|------|------|------|
| base_price | 정수 | 상점 기본 가격 |
| weight | 실수 | 무게 |
| stackable | 불리언 | 중첩(겹치기) 가능 여부 |
| max_stack | 정수 | 최대 중첩 수량 |
| level_required | 정수 | 사용/착용 최소 레벨 |
| class_required | 문자열 | 필요 클래스 ID |
